### < 문제 >
인류가 전쟁을 없애지 않는다면 전쟁이 인류를 없앨 것이다.
하루빨리 전쟁을 없애기 위해 동분서주하고 있는, 세계에서 유일한 중립국의 수석 정보요원인 당신에게 두 번째 지령이 내려왔다.
이번에는 조금 더 자세한 분석을 위해 도시 단위로 데이터를 수집해야 한다.
단, 다음과 같은 조건을 만족해야 한다.
<br><br>
A와 B가 같은 국가 소속 도시이고 B와 C도 그러하다면 A와 C도 같은 국가 소속이다.
어떤 도시 A와 B가 같은 국가에 속해 있다고 주어지고 B가 국가 1에 속해 있으면 A도 국가 1에 속해 있다.
기존과 충돌하는 정보가 주어지면 나중의 것을 우선시한다.
도시와 그 소속국에 대한 정보가 입력되면 원하는 출력을 만들어내는 프로그램을 만들자.
<br><br>
##### 입력
첫째 줄에는 도시의 개수 N이 주어진다. (N은 1 이상 100,000 이하)
다음 줄에는 같은 국가에 속한 도시에 대한 정보의 수 M이 주어진다.
다음 줄부터 M개 줄엔 같은 국가에 속한 도시 두 개가 공백으로 분리되어 주어진다.
각 정보는 한 줄에 하나씩 제시된다.
각 정보는 A B의 형식으로 이루어진다. 이는 도시 A와 B가 같은 국가에 속했음을 의미한다.
이후 도시와 국가의 소속 관계를 나타내는 정보의 수 C가 주어진다.
다음 줄부터 C개 줄엔 도시와 국가에 종속 관계가 주어진다.
각 정보는 A B의 형식으로 이루어진다. 이는 도시 A가 국가 B에 속한다는 의미이다.
M, C, 그리고 국가의 이름은 int 범위 이내의 자연수이다.
<br><br>
##### 출력
1번 도시부터 N번 도시까지 어떤 국가에 속해 있는지를 공백으로 분리하여 출력한다.
단, 소속 국가를 알 수 없을 경우 0을 출력한다.
<br><br>
입력 예시 <br>  
3<br>
2<br>
1 2<br>
2 3<br>
1<br>
3 1
<br><br>
출력 예시<br>
1 1 1

<br><br>
### < 풀이 >

Union-Find를 이용해서 같은 국가에 속하는 도시들을 합쳐주면 됩니다. 하지만, 도시의 개수가 크기때문에 시간제한을 맞추기 위해 경로압축을 시켜주어야한다. <br>

{% highlight c roudge linenos %}

  #include <stdio.h>

  int ary[100001],cou[100001];

  int find(int i)
  {
  	if(ary[i]==i) return i;
  	else return ary[i]=find(ary[i]); // 경로 압축
  }

  void connect(int a, int b)
  {
  	a=find(a);
  	b=find(b);
  	if(a>b) ary[a]=b;
  	else ary[b]=a;
  }

  int main()
  {
  	int a,b,c,n,m,i;
  	scanf("%d",&a);
  	for(int i=0; i<a; i++){
  		ary[i]=i;
  	}
  	scanf("%d",&b);
  	for(i=0; i<b; i++)
  	{
  		scanf("%d %d",&m,&n);
  		connect(m,n);
  	}
  	scanf("%d",&c);
  	for(i=0; i<c; i++)
  	{
  		scanf("%d %d",&m,&n);
  		cou[find(m)]=n;
  	}
  	for(i=1; i<=a; i++)
  	printf("%d ",cou[find(i)]);
  }
