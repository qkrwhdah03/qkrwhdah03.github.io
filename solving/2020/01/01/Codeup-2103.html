Codeup 2103 배수(Hell)

by qkrwhdah03

2020. 05. 02
in Solving

문제 출처

< 문제 >

하나의 자연수 n (n은 2000000이하)를 입력받아 0과 1로 이루어진 최소의 n배수를 출력하시오.

< 풀이 >

사실 처음에 굉장히 고민을 많이 했다. 이 문제와 같이 단순히, n 배수 하나를 출력하는 것이 었다면 1, 11, 111, … 들의 숫자를 n개 만들어주어 n으로 나눈 나머지들을 구하고, 이 n개의 숫자들 중 n배수가 있다면 해당 수를 출력하면 되고, n배수가 없다면 비둘기 집의 원리에 의해 n개의 숫자들 중 n으로 나눈 나머지가 같은 쌍이 적어도 1쌍 존재하기 때문에 그 둘을 빼주어 출력하면 된다.

하지만, 이 문제는 최소의 n배수를 출력하는 문제이다. 그렇기 때문에 탐색을 어떻게 할지에 대해 고민했고, 10^k 꼴의 수들을 n으로 나눈 나머지를 r[k]의 배열에 저장해주고, 이 r[k]들의 합을 bit masking으로 표현하여 문제를 해결했다. 예를 들어, r[1]+r[3]의 경우는 2진수로 000000…001010(2)의 형태로 저장해주었다. 즉, 오른쪽에서 k+1위치의 비트가 r[k]가 더해졌는지의 여부를 나타내는 것이다. 이 방식을 이용해서 다음 코드와 같이 작은 자리 수부터 r[k]의 모든 합들을 구하면서 탐색해가면 O(kn)의 시간복잡도로 문제를 해결할 수 있다. (여기서, k는 구하려는 수의 자리수이다.)

< 코드 >

  #include <iostream>
  #define N 100
  using namespace std;
  typedef long long ll;
  int n,r[N]; // r[k] =10^k %n
  ll res[2000001]={0,}; // res[k]=m 나머지가 k인 조합을 m의 bit로 나타냄

  void f(ll tmp){ //2 진수를 역수로 출력
    if(tmp==0) return;
    f(tmp/2);
    cout<<tmp%2;
  }

  int main(){
    ios_base::sync_with_stdio(false);cin.tie(NULL);cout.tie(NULL);
    cin>>n;
    if(n==1){
      cout<<1;
      return 0;
    }
    for(int i=0; i<N; ++i){
      if(i==0) r[0]=1;
      else r[i]=(10*r[i-1])%n;
    }
    for(int i=0; i<N; ++i){
      ll tmp=1LL<<(i+1);
      if(res[r[i]]==0 || res[r[i]] > tmp) res[r[i]]=tmp;
      for(int j=0; j<n; ++j){
        if(res[j] && ((res[j]>>(i+1))&1LL) ==0){
          int idx=(j+r[i])%n;
          ll tmp=res[j];
          tmp=tmp|(1LL<<i+1);
        if(res[idx]==0 || res[idx]>tmp) res[idx]=tmp;
        }
      }
      if(res[0]!=0) break;
    }
    ll tmp= res[0]>>1;
    f(tmp); // 출력
  }
